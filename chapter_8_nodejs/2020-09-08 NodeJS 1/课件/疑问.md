## nodejs和javascript的关系

拉丁字母：a，b，c  => ECMAScript

英语 => javascript，ECMAScript + DOM + DOM ……（浏览器，页面元素）yes，document，window (chrome -> v8)

汉语拼音 => Node.js，ECMAScript + 操作（windows，Linux）+ FileSystem + Net + Memory + ……。Nodejs编译器（类似浏览器的执行环境）-> v8
